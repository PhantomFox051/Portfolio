# Проекты

## Список проектов

- **Tank Game** — танк с башней, которая поворачивается за мышкой. Управление WASD, поворот башни — мышка. (raylib)

- **Flask Weight Service** — веб-сервер: по строке из запроса считает вес символов по данным из файла input.txt. (Flask)

- **University API** — поиск университетов России по названию через внешнее API. (requests)

- **Arcade Ball** — аркадный мячик, отскоки от стен. (arcade)

- **Pygame Examples** — игры на pygame: мяч с отскоками, кликер по кругу. (pygame)

- **JSON Manager** — консольная утилита для работы с JSON: создание, чтение, запись, копирование. (стандартная библиотека)

- **CSV Analyzer** — чтение CSV-файла с данными, расчёт максимумов и среднего. (стандартная библиотека)

## Как запускать

В каждой папке с проектом есть файл requirements.txt. Чтобы запустить проект:

```bash
pip install -r requirements.txt
python main.py
